## Start frontend
1. ### `$ cd client`
2. ### `$ yarn install`
3. ### `$ yarn start`

## Start backend
1. Create Python virtual environment
2. ### `$ pip install -r requirements.txt`
3. Setup Postgres locally or in a Docker container
    ### `USER: postgres`
    ### `PASSWORD: postgres`
    ### `HOST: localhost`
    ### `DATABASE: postgres`
5. ### `$ export FLASK_ENV=development`
4. ### `$ python app.py`
